# SINGLE SHIP SURVIVAL - CONTACT B

単一HTMLで作成したゲームプロトタイプです。

## 概要

単艦で夜間巡航中、未確認接触に対して「生き残る」ことを目的とした判断ゲームです。
接触の正体は「敵艦」か「民間船」。プレイヤーは限られた情報の中で、

* 静観するか
* 能動的に索敵するか
* 先制攻撃するか
* 時間経過で様子を見るか

を選択し続けます。

誤った判断は即被弾や誤射につながり、正解が明示されない中での意思決定が求められます。
戦闘アクションではなく「不完全情報下の判断」と「リスク管理」を体験するミニマルな戦術シミュレーションです。

## 遊び方

1. ページを開く
2. レーダーとログを確認しながら選択肢を選ぶ
3. 接触の正体を見極めつつ、生き残る判断を続ける
4. エンディングまで進める

## 目的

このプロトタイプでは以下を検証することを目的としています。

* 最小限のUIで戦術的な判断ゲームが成立するか
* 「撃つ / 待つ / 観察する」のシンプルな選択だけで緊張感を生み出せるか
* グラフィック（艦の存在感）とミニマルなゲーム性の両立が可能か
* 正解が明示されない状況におけるプレイヤーの意思決定体験の質
